## Overview

GeistOS is a lightweight, terminal-based operating system focused on simplicity and clarity.  
It is designed as a minimal environment that runs entirely in the command line and avoids unnecessary complexity.

The project explores how far a basic system can go while remaining understandable and easy to extend.

---

## Features

- Simple command-line interface  
- Minimal system structure  
- Lightweight and fast  
- Easy to modify and expand  
- No graphical overhead
